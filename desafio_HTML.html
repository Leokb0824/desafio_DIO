O HTML é a base dos sites que conhecemos hoje

A internet está repleta de sites e aplicativos, e a cada dia novos sites surgem, sites de e-commerce, bancos, redes sociais, informativos entre outros milhares de temas que são abordados por esses endereços web, mais o que esses sites têm em comum mesmo sendo cada um deles voltado a públicos e utilidades diferentes?
A resposta é a sigla composta por 4 letras HTML, O significado de HTML é HyperText Markup Language ou para nós brs Linguagem de Marcação de Hipertexto.
Diferente do que muitos acreditam, o HTML não é uma linguagem de programação, mas ela está presente na vida de todos os programadores. O HTML é baseado em Tags que são lidas pelos browsers ou como são popularmente conhecidos Navegadores, esses navegadores conseguem ler as tags e definir as funções que cada uma delas tem dentro de seu site.
O HTML junto do CSS e JAVASCRIPT formam a tríade para a criação para a maioria dos sites existentes hoje.

O que são Tags ?

Voltar

O HTML, como já foi dito anteriormente, é formado por Tags. Essas tags são responsáveis por fornecer aos navegadores a estrutura do site que está sendo criado através dele. Imagine dessa forma: um site hoje é criado em sua maioria por 3 elementos base que já os denominamos de triade, composta por HTML, CSS, JAVASCRIPT(JS). Fazendo uma analogia a um corpo humano, o HTML é o esqueleto, o CSS seria a pele e o JS seria o cérebro que traz a inteligência e que faz com que esse corpo possa se mover.
O HTML, sendo como o esqueleto do site, pense nas tags como sendo cada osso desse esqueleto que deve se encaixar ao outro sem ultrapassar o limite de cada um.
As tags são construídas com a abertura e fechamento delas. A sintaxe para abertura é o símbolo "<" seguido do comando, como exemplo "body", e o símbolo ">", ficando dessa forma "< body >". Após a abertura da tag vem o conteúdo dela e, por fim, o seu fechamento, que é feito dessa forma "< /body >". Note que para o fechamento é acrescentado apenas "/" antes do comando, e assim é feito o fechamento da tag.
Existem diversas tags no HTML, tags para iniciar um texto como a tag "p", tag de imagem como "img", tags de corpo de site como a "body" e a "head", e diversas outras que compõem um site.
